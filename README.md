# Kooboo-CLI

## 使用

### 创建项目

```bash
npx @kooboo_cli/cli create
```

### 初始化

```bash
cd 项目目录

# pnpm
pnpm install

pnpm dev:init # 将本地代码初始化推送到远端

# npm
npm install
npm run dev:init # 将本地代码初始化推送到远端
```

### 开发

该命令会运行监听脚本, 即本地代码发生变化时, 会自动同步到远端

```bash
# pnpm
pnpm dev

# npm
npm run dev
```
